Some web pages related to Angband

  • The Official Angband Home Page is the "official" Angband page.
  • The Unofficial Angband Home Page is a very nice page about Angband, and includes up to date copies of the FAQ.
  • The Early History of Moria page is maintained by Robert Alan Koeneke, the creator of Moria.
  • The Pits of Angband page is part of a collection of pages describing various games which are based in the Tolkien Universe.
  • The Angband Home Page is something I ran across recently.
  • The Moria Page describes the predecessor to Angband.
  • The What About Gamma page describes the technical details behind "Gamma Correction", a problem that Angband fails to handle correctly.
  • There is a new Winner Posts page.
  • There is a new Dungeon Cam page.
  • The Linux Angband Compilation page was recently added.
  • Here is another Angband Page.
  • Here is another Angband Page.

Some ftp sites related to Angband

  • The clockwork.dementia.org ftp site contains the most up to date copies of the source and executables for Angband.
  • The ftp.cis.ksu.edu ftp site is the official "mirror" site.
  • The ftp.sunet.se ftp site is a European "mirror" site.

Some newsgroups related to Angband

  • The rec.games.roguelike.angband newsgroup is a good place to visit for hints about playing Angband and for discussion about the latest version from a programmers point of view.
  • The rec.games.roguelike.moria newsgroup talks about the game from which Angband was developed.
  • The rec.games.roguelike.rogue newsgroup talks about one of the first "roguelike" games.
  • The rec.games.roguelike.announce newsgroup contains announcements about the various roguelike games.
